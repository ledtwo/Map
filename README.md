
## echarts4 中国地图下钻至市、县、乡镇级

#### 1、使用方法

启动本地服务，再打开index.html
> 注意不可浏览器直接打开源文件，需要初始化加载地图，所以需要启动本地服务

示例demo：https://m.glinteah.com/map
中国 > 河南 > 许昌市 > 各县乡镇

![效果](https://m.glinteah.com/static/preview.jpg)

>全国地图、世界地图、各省份地图数据JSON可查看map文件夹
